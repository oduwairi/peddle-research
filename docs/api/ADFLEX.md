# Adflex API Documentation

Learn how to use the Adflex API to access millions of ads data from top platforms, with guides, filters, endpoints, and integration examples for developers.

Version: 1.0.0

## Servers

Server
```
https://api.adflex.io/api/{version}
```

Variables:
- `version`
Default: "v1"
Enum: "v1"

## Download OpenAPI description

[Adflex API Documentation](https://doc.adflex.io/_bundle/openapi.yaml)

## Facebook

### Get search filters

 - [GET /filters/facebook/search](https://doc.adflex.io/openapi/facebook/facebooksearchfilters.md)

### Get interests items

 - [GET /facebook/filter/interests/items](https://doc.adflex.io/openapi/facebook/facebookfilterinterestsitems.md)

### Get fanpage categories items

 - [GET /facebook/filter/owner_categories/items](https://doc.adflex.io/openapi/facebook/facebookfilterfanpagecategoriesitems.md)

### Search in facebook ads

 - [POST /facebook/ads/search](https://doc.adflex.io/openapi/facebook/facebookadssearch.md)

### Get ad's details

 - [GET /facebook/ads/{ad}](https://doc.adflex.io/openapi/facebook/facebookaddetail.md)
